# Development setup

## Node installation

Install [`nvm`](https://github.com/nvm-sh/nvm#installing-and-updating).

```
nvm install $(< .nvmrc)
```

## Install dependencies

```
yarn install
```

## Compiles and hot-reloads for development

```
yarn electron:serve
```

## Run your unit tests

```
yarn test:unit
```

## Lints and fixes files

```
yarn lint
```
